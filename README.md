# assingment tracking app 
this app was made to track all of your school work in one place, made to organize your assignments and keep you on track so you never turn in a assignemnt late. 


## Features
- a light and dark mode to suit your prefrences
- notes on each assignment so you remember where the resources may be, what to study, ect.
- add due dates to keep you on track and make sure its not late.
- when each subject runs out of tasks it auto deletes.

## Installation
Under the releases section, download the latest installer.
> Note, windows smart-screen may trigger!

## Manual Building
Download the code, or clone the repo.
Then run the following commands

Run in development mode:
```bash
cd '.\Desktop app\'
npm start
```

Build:
```bash
npm run dist
```
